Background

In this activity we will be using modulo (%)! It is often spoken about as the remainder, the complement to integer division. We often use it to find if something is even or odd, by inspecting if x % 2 == 0 for even and x % 2 == 1 for odd. Another way to think of it is to convert linear change into cyclic change. Consider how it acts on this linearly increasing list: nums = [0, 1, 2, 3, 4, 5, 6, 7, 8] print([x % 3 for x in nums]) >> [0, 1, 2, 0, 1, 2, 0, 1, 2] It turns 0..8 into (0, 1 , 2) repeating!
